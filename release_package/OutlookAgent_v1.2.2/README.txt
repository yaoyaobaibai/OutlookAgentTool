# Outlook Agent v1.2.2
## Outlook 邮件监控 & PDF 合并工具

### 安装说明

1. 将 OutlookAgent.exe 和 PDFMergeTool.exe 放在同一目录下
2. 双击运行 OutlookAgent.exe
3. 程序自动检测同目录下的 PDFMergeTool.exe（无需手动配置）
4. 确保 Outlook 经典版已打开并登录
5. 注意：不支持 "New Outlook"，请使用经典版 Outlook

### 使用方法

**方式一：自动监控**
1. 启动 OutlookAgent.exe
2. 程序会自动监控 Outlook 邮件
3. 检测到包含关键字的邮件时，会弹出通知
4. 选择需要合并的附件，点击"确认合并"
5. 合并完成后的 PDF 保存在 用户目录\merged_output\

**方式二：手动上传 .msg 文件**
1. 在主界面点击"选择 .msg 文件"按钮
2. 选择 Outlook 导出的 .msg 文件
3. 程序自动提取附件并合并为 PDF
4. 适用于处理历史邮件或导出的邮件文件

### 配置文件位置

- 配置文件: 用户目录\outlook_agent_config.json
- 输出目录: 用户目录\merged_output\
- 临时文件: 用户目录\outlook_agent_temp\
- 日志文件: 用户目录\outlook_agent_logs\
- 诊断包: 用户目录\merged_output\crash_dumps\（出问题时自动保存）

### 邮件监控设置（可在设置窗口中修改）

**监控模式：**
- 时间监控：监控最近 N 分钟内的所有邮件
- 数量监控：监控最近 N 封邮件
- 两者结合：最近 N 封 + 时间过滤

**默认设置：**
- 监控模式：时间监控
- 时间范围：60 分钟
- 数量限制：50 封

### 默认监控关键字

- 合同审批
- 文件合并
- PDF合并
- 附件处理
- 合并附件

可在设置窗口中自定义关键字。

### 支持的文件格式

- PDF 文件: .pdf
- Word 文档: .docx, .doc
- Excel 表格: .xlsx, .xls
- 文本文件: .txt
- 图片文件: .png, .jpg, .jpeg, .gif, .bmp
- Outlook 邮件: .msg（自动提取附件并合并）
- 压缩包: .zip（自动解压并合并内部文件）

### .msg 文件处理说明

**手动上传的 .msg 文件**（通过"选择 .msg 文件"按钮）：
- 自动提取邮件中的所有附件
- 合并到最终PDF

**监控到的新邮件中的 .msg 附件**：
- 自动提取附件并合并到PDF

**嵌入的 .msg 邮件附件**（邮件附件中包含的 .msg）：
- 只提取嵌入邮件的正文和邮件信息（发件人、主题、日期）
- 不处理嵌入邮件内的附件

### 系统要求

- Windows 10/11
- Microsoft Outlook 经典版（桌面版），已安装并登录
- ⚠️ 不支持 "New Outlook for Windows"（新版 Outlook）
- 无需安装 Python 或其他运行环境

### 首次使用提示

- 启动后程序会在任务栏显示窗口，同时后台开始监控邮件
- 检测到含有关键字的邮件时，会自动弹出确认窗口
- 如果没有匹配的邮件，程序静默运行，不会弹窗
- 合并后的 PDF 保存在：C:\Users\你的用户名\merged_output\
- 如果部分文件损坏，其余文件仍会正常合并，不会全部失败
- 如需修改关键字或监控范围，点击"设置"按钮

### 常见问题

Q: 提示无法连接 Outlook
A: 确保 Outlook 正在运行（经典版，非 New Outlook），并已登录邮箱账户

Q: 提示"下载 Outlook 2016"
A: 你用的是 New Outlook，请切回经典版（Outlook 右上角关闭 "New Outlook" 开关）

Q: 合并失败
A: 检查日志文件 用户目录\outlook_agent_logs\ 查看详细错误。也可点击主界面"查看日志"按钮快速打开。

Q: 文件较多时合并很慢或卡住
A: 正常现象。程序会根据文件数量自动调整等待时间，每个文件约预留 1 分钟。如果文件特别多（超过 10 个），建议分批处理。

Q: 合并后提示"部分文件无法处理"
A: 个别文件可能已损坏，其余文件仍会正常合并。出问题的文件信息会记录在日志中，不影响其他文件。

Q: Word 文件转换后格式丢失，或日志显示""文件被信任中心阻止""
A: 程序会自动绕过企业 Word 安全策略。如仍有问题：
   - 中文：Word → 文件 → 选项 → 信任中心 → 信任中心设置 → 文件阻止设置 → 取消勾选 ".docx"
   - 英文：Word → File → Options → Trust Center → Trust Center Settings → File Block Settings → uncheck ".docx"
   如果选项灰色（被 IT 锁定），联系管理员。

Q: 如何反馈问题
A: 点击主界面"导出诊断包"按钮，将生成的压缩包发给开发者，里面包含了日志和配置信息，方便快速排查问题。
