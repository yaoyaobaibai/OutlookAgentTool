================================================================================
                    表单自动填充工具 - FormFiller
                         使用说明书
================================================================================

一、程序位置
------------
可执行文件：dist\FormFiller.exe
快速启动：双击 "启动表单填充工具.bat"

二、功能特性
------------
✓ 从 Excel 文件读取表单字段值
✓ 自动填充网页表单
✓ 支持多个附件上传管理
✓ 支持 Chrome、Edge、Chromium 浏览器
✓ 可保存字段配置和附件配置

三、使用步骤
------------

【步骤 1】准备 Excel 数据文件
创建 Excel 文件，包含需要填充的字段列名，例如：
  | Proposal # | Cust Ref. No | Proposal/Contract Value | ... |
  | P12345     | CR-2024-001  | 500000                  | ... |

【步骤 2】配置表单字段
1. 运行程序 FormFiller.exe
2. 输入目标网址（登录页面 URL）
3. 点击"添加字段"按钮
4. 填写：
   - 字段名称：必须与 Excel 列名一致
   - CSS 选择器：网页元素的 CSS 选择器
   - 默认值：可选
5. 重复添加所有需要的字段

【步骤 3】添加附件（可选）
1. 在"附件上传管理"区域点击"添加附件"
2. 选择 Category（附件类别）
3. 点击"浏览"选择文件
4. 填写 Description（描述）
5. 点击"确定"

【步骤 4】执行自动填充
1. 选择浏览器类型（推荐 Chrome）
2. 点击"浏览"选择 Excel 文件
3. 点击"启动填充"
4. 程序会打开浏览器并访问登录页面
5. 手动输入用户名密码登录
6. 程序自动填充表单和上传附件

四、推荐配置（基于实际网页）
------------------------------
字段名称                  CSS 选择器
---------------------------------------------------------------
Proposal #               #ctl00_ContentPlaceHolder1_txtProposalNo
Cust Ref. No             #ctl00_ContentPlaceHolder1_txtCustRefNo
Proposal/Contract Value  #ctl00_ContentPlaceHolder1_txtContractValue
Currency Code            #ctl00_ContentPlaceHolder1_ddlSelPriceCurrCode
Date of Award            #ctl00_ContentPlaceHolder1_dtDateofAward_txtDate

五、附件上传配置
----------------
程序会自动按顺序处理附件列表中的每个附件：
1. 选择 Category 下拉框
2. 上传文件
3. 填写 Description

如果网页表单的附件上传区域选择器不同，可以手动添加字段：
  字段名称：Attachment 1 - Category
  CSS 选择器：#实际的 category 选择器
  默认值：Proposal Document

  字段名称：Attachment 1 - File
  CSS 选择器：file:#实际的文件上传选择器
  默认值：C:\path\to\file.pdf

  字段名称：Attachment 1 - Description
  CSS 选择器：#实际的 description 选择器
  默认值：项目建议书

注意：文件上传字段的选择器需要添加 "file:" 前缀

六、配置文件
------------
form_config.json - 保存表单字段配置
attachment_config.json - 保存附件配置

这些文件会自动创建和更新，无需手动编辑。

七、常见问题
------------

Q1: 程序打开后无法找到浏览器？
A: 确保已安装 Chrome 或 Edge，或在程序中指定正确的浏览器路径。

Q2: 填充失败，提示找不到元素？
A: 检查 CSS 选择器是否正确。可以使用浏览器开发者工具（F12）查看元素。

Q3: 附件上传失败？
A: 检查文件路径是否正确，确保文件存在。

Q4: Excel 读取失败？
A: 确保 Excel 文件格式正确，列名与字段名称完全匹配。

Q5: 如何修改已保存的配置？
A: 在程序主界面选择字段或附件，点击"编辑"按钮进行修改。

八、技术支持
------------
如遇到问题，请检查：
1. 控制台输出的详细日志
2. 浏览器开发者工具中的元素选择器
3. Excel 文件格式是否正确
4. 网络连接是否正常

九、注意事项
------------
1. 首次运行可能需要安装浏览器驱动
2. 登录页面需要手动输入（防止验证码）
3. 使用绝对路径避免文件找不到
4. 定期备份配置文件
