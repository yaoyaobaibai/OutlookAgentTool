## 📋 程序已更新的功能

### ✅ 新增自动登录功能
程序现在支持完全自动化流程：
1. **自动登录** - 输入用户名和密码后自动登录

## 📦 附件上传配置

### 方式 1：使用程序界面添加（推荐）
1. 点击"添加附件"按钮
2. 选择 Category
3. 浏览选择文件
4. 填写 Description
5. 点击"确定"

### 方式 2：手动编辑配置文件
编辑 `attachment_config.json`：
```json

  "attachments": [
    {
      "category": "Proposal Document",
      "file_path": "C:\\Documents\\proposal.pdf",
      "description": "项目建议书"
    },
    {
      "category": "Contract",
      "file_path": "C:\\Documents\\contract.docx",
      "description": "合同文件"
    }
  ]
}
```

### 支持的 Category 值
- Proposal Document
- Contract
- Supporting Document
- Technical Specification
- Financial Document
- Other

## 🚀 完整使用流程

### 步骤 1：准备 Excel 数据
```excel
创建文件：proposal_data.xlsx
包含列：Proposal #, Cust Ref. No, Proposal/Contract Value, Currency Code, Date of Award
```

### 步骤 2：配置表单字段
在程序中依次添加：
```
1. 字段名称：Proposal #

2. 字段名称：Cust Ref. No
```

## ⚠️ 常见问题

### Q1: 登录失败？
**A**: 检查：
- 用户名和密码是否正确
- 登录页面是否有验证码（需要手动处理）
- 网络连接是否正常

### Q2: 找不到登录表单？
**A**: 程序会尝试多个常见选择器。如果都失败，需要手动登录一次，然后查看浏览器开发者工具（F12）找到正确的选择器。

### Q3: 填充字段失败？
**A**: 检查：
- CSS 选择器是否正确
- Excel 列名是否与字段名称完全匹配
- 字段是否可见（没有被遮挡）

### Q4: 附件上传失败？
**A**: 检查：
- 文件路径是否正确（使用绝对路径）
- 文件是否存在
- 网页表单的附件上传区域选择器是否正确

### Q5: 如何禁用自动登录？
**A**: 取消勾选"自动登录"复选框，程序会等待手动登录。
