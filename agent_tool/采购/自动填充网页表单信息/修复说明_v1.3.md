### 问题 1: Selling Price Currency Code 下拉框 ❌ → ✅

**问题描述**:
```
Locator.fill: Error: Element is not an <input>, <textarea> or [contenteditable] element
```

**原因**: 
- 程序使用 `fill()` 方法处理下拉框 `<select>` 元素
- 应该使用 `select_option()` 方法

**修复方案**:
```python
# 检测元素类型
tag_name = element.evaluate('el => el.tagName.toLowerCase()')

if tag_name == 'select':
    # 下拉框使用 select_option
    element.select_option(value)
    print(f"✓ 已选择：{value}")
elif tag_name == 'input' or tag_name == 'textarea':
    # 输入框使用 fill
    element.fill(value)
    print(f"✓ 已填充")
else:
    # 其他类型尝试 JavaScript 设置
    element.evaluate(f'el => {{ el.value = "{value}"; el.dispatchEvent(new Event("change")); }}')
```

### 问题 3: Priming Project Manager 弹窗搜索 ❌ → ✅

**问题描述**:
- 需要点击放大镜按钮打开搜索弹窗
- 在弹窗中输入 LoginID（如：lin.zu）
- 点击 Search 搜索
- 选择第一个结果
- 点击 Select 确认

**修复方案**:
```python
# 1. 点击放大镜按钮
pm_btn = page.locator('input[id*="ucEmpSearch"][id*="txtUserName"]').first
with page.expect_popup() as popup_info:
    pm_btn.click()

popup = popup_info.value
popup.wait_for_load_state('networkidle')

# 2. 输入 LoginID
login_input = popup.locator('#txtOAID')
login_input.fill('lin.zu')

# 3. 点击 Search
search_btn = popup.locator('input[type="submit"][value*="Search"]').first
search_btn.click()
popup.wait_for_timeout(2000)

# 4. 选择第一个结果
first_row = popup.locator('table tr:nth-child(2)').first
first_row.click()

# 5. 点击 Select 确认
select_btn = popup.locator('input[type="submit"][value*="Select"]').first
select_btn.click()
```

**LoginID 来源**:
- 从字段配置的 `value` 属性读取
- 可以是固定值（在字段配置中填写）
- 也可以从 Excel 读取

### 问题 4: 附件上传功能 ❌ → ✅

**问题描述**:
- Category 是下拉框，没有成功选择
- 文件没有上传
- Description 没有填写

**修复方案**:
```python
# 对每个附件执行 3 步操作
for i, attachment in enumerate(attachments, 1):
    
    # [1/3] 选择 Category（下拉框）
    category_patterns = [
        f"select[id*='ddlAttachmentCategory_{i}']",
        f"select[name*='AttachmentCategory_{i}']",
        f"select[name*='category_{i}']",
    ]
    for pattern in category_patterns:
        cat_elem = page.locator(pattern).first
        if cat_elem.count() > 0:
            cat_elem.select_option(attachment['category'])
            break
    
    # [2/3] 上传文件
    file_patterns = [
        f"input[type='file'][id*='FileUpload_{i}']",
        f"input[type='file'][name*='file_{i}']",
    ]
    for pattern in file_patterns:
        file_elem = page.locator(pattern).first
        if file_elem.count() > 0:
            file_elem.set_input_files(attachment['file_path'])
            break
    
    # [3/3] 填写 Description
    desc_patterns = [
        f"textarea[id*='txtAttachmentDesc_{i}']",
        f"textarea[name*='desc_{i}']",
    ]
    for pattern in desc_patterns:
        desc_elem = page.locator(pattern).first
        if desc_elem.count() > 0:
            desc_elem.fill(attachment['description'])
            break
```

**支持的 Category 值**:
- Proposal Document
- Contract
- Supporting Document
- Technical Specification
- Financial Document
- Other

## ⚠️ 常见问题

### Q1: 日历页面打不开？
**A**: 
- 检查浏览器是否允许弹窗
- 确认网络连接正常
- 增加等待时间：`page.wait_for_timeout(3000)`

### Q2: Project Manager 搜索不到用户？
**A**:
- 检查 LoginID 是否正确
- 确认用户在职且有系统权限
- 尝试使用其他 LoginID 测试

### Q3: 附件上传失败？
**A**:
- 检查文件路径是否正确（使用绝对路径）
- 确认文件存在
- 查看页面 HTML 结构调整选择器

### Q4: 下拉框选择失败？
**A**:
- 检查选项值是否正确（区分大小写）
- 确认下拉框已加载完成
- 查看控制台日志中的元素类型
