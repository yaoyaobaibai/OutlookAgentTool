# FormFiller - 打包和使用说明

## ⚡ 快速开始（立即可用）

### 方式 1：直接运行 Python 脚本
```bash
# 双击运行
运行程序.bat

# 或命令行运行
python form_filler.py
```

**优点**：
- ✅ 立即可用，无需等待打包
- ✅ 方便调试和修改代码
- ✅ 占用空间小

**缺点**：
- ❌ 需要安装 Python
- ❌ 需要安装依赖包

## 📦 打包为 EXE（可选）

### 方式 2：使用打包脚本

```bash
# 双击运行（需要 5-10 分钟）

```

**优点**：
- ✅ 独立运行，无需 Python
- ✅ 方便分发给其他人
- ✅ 更简洁的使用方式

**缺点**：
- ❌ 打包时间长（5-10 分钟）

## 📋 使用流程

### 1. 运行程序
```bash
# 双击
运行程序.bat
```

### 2. 配置登录信息
- **登录网址**：`https://csmstest.ncs.com.sg/UAT/`
- **用户名**：你的登录账号
- **密码**：你的登录密码
- ✅ 勾选"自动登录"

### 3. 添加表单字段

**必填字段（5 个）**：

| 字段名称 | CSS 选择器 |
|---------|-----------|
| Proposal # | `#ctl00_ContentPlaceHolder1_txtProposalNo` |
| Cust Ref. No | `#ctl00_ContentPlaceHolder1_txtCustRefNo` |
| Proposal/Contract Value | `#ctl00_ContentPlaceHolder1_txtContractValue` |
| Selling Price Currency Code | `#ctl00_ContentPlaceHolder1_ddlSelPriceCurrCode` |
| Date of Award | `#ctl00_ContentPlaceHolder1_dtDateofAward_txtDate` |

**添加方法**：
1. 点击"添加字段"
2. 填写字段名称和 CSS 选择器
3. 点击"确定"
4. 重复以上步骤添加所有字段

### 4. 准备 Excel 文件

Excel 文件应包含以下列：

```excel
| Proposal # | Cust Ref. No | Proposal/Contract Value | Currency Code | Date of Award |
|------------|--------------|------------------------|---------------|---------------|
| P12345     | CR-2024-001  | 500000                 | USD           | 01/15/2024    |
```

**注意**：
- 列名必须与字段名称**完全一致**
- 日期格式：MM/DD/YYYY
- 货币代码：USD, SGD, EUR 等

### 5. 添加附件（可选）
1. 点击"添加附件"
2. 选择 Category
3. 浏览选择文件
4. 填写 Description
5. 点击"确定"

### 6. 启动填充
1. 选择浏览器（推荐 Chrome）
2. 点击"浏览"选择 Excel 文件
3. 点击"启动填充"
4. 程序会自动完成所有操作

---

## 🎯 自动化流程

程序会自动执行以下步骤：

1. ✅ **打开浏览器**并访问登录页面
2. ✅ **自动登录**（填写用户名、密码、点击登录）
3. ✅ **导航**到 Create Proposal Group 页面
4. ✅ **填写 Proposal #** 并点击 GET CRM INFO
5. ✅ **填充所有表单字段**（从 Excel 读取）
6. ✅ **上传附件**（如果已添加）
7. ⏸️ **停止**（等待你检查后手动点击 Create）

## ⚠️ 常见问题

### Q1: 登录失败？
**A**: 检查用户名密码是否正确，是否有验证码（需要手动处理）

### Q2: 找不到元素？
**A**: 检查 CSS 选择器是否正确，使用 F12 开发者工具验证

### Q3: Excel 读取失败？
**A**: 确保列名与字段名称完全匹配（区分大小写）

### Q4: 附件上传失败？
**A**: 检查文件路径是否正确（使用绝对路径）

### Q5: 打包失败？
**A**: 
- 确保已安装 PyInstaller: `pip install pyinstaller`
- 确保有足够的磁盘空间
- 关闭杀毒软件（可能误报）
- 重新运行 `执行打包.bat`
